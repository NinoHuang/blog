# 体脂率计算公式如何计算

每个人对自己身材的形容几乎是以胖和瘦来区别，当发现自己胖了，那么就努力让自己减肥，如果发现自己瘦了，那么就可以增加饮食摄入量进行增肥。其实不论是减肥还是增肥，最好的身材是要根据个人的体脂率去进行计算，很多人不清楚自己的体质率是多少，也不知道体脂率计算公式是如何的。

# 体脂率的计算公式

体脂率计算公式是：（身体脂肪总重量÷体重）×100%，简单的说就是反应一个人体脂含量占总体重的百分比。一个人的体脂率会直接反应出这个人的身材如何，是否已经处于肥胖阶段，体脂率越低那么肌肉就会越强壮。

体脂率不同的人身材表现完全不同：一个人想要保持正常的状态，那么他的体脂率必须要超过3%，如果只是在5~7%之间，那说明这个人属于一个健身爱好者，身体的肌肉比较明显，尤其是胳膊与腿，都可以清晰可见的血管，腹部肌肉也较为明显。很少有人可以锻炼到这个程度，除非他是一个极大的健身爱好者，有极其规律的健身运动习惯。

# 保持体脂率低有什么效果

若体脂率达到8~10%那么通常腹部会有肌肉感。若体脂率在11~12%之间，肌肉轮廓几乎还在，只是棱角没那么清晰。当体脂率达到16~19%，这基本已经含有多余的脂肪，尤其是臀部和腹部。当腰部，背部，腿部都有脂肪出现时，这基本已经处于体脂率在20~24%之间。

通过以上的内容介绍可见，一个人的体脂率直接会反映出这个人的身材状况如何，身体的脂肪含量以及肌肉含量的数据。如果不了解体脂率，当阅读过以上内容相信已经有了答案，为了达到最完美的身材，要了解体脂率计算公式，学会计算自己的体脂率，这样才可以知道通过怎样的锻炼强度帮助自己达到满意的身材。
